# 프로젝트 테이블 설계
## 고민한 부분
+ 게시글이 이미지를 여러개 갖는 경우 어떻게 테이블을 설계할 것인가
+ 사용자가 게시글에 좋아요 와 싫어요를 누르는 경우 어떤 사용자가 어떤 게시글에 눌렀는지 판단하기 위한 테이블 구조에 대한 고민
+ 채팅 기능 - 게시글에 대한 질문을 채팅으로 남기는 경우 채팅방과 게시글 테이블의 구조?
  + 게시글이 100개일때 100개의 채팅방을 만들것인가?
  + 게시글 여러개 채팅방 하나
